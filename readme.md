<h1> Turtle Sim </h1>

<b>Keyboard commands:</b>

Arrow keys and WASD: move turtle

Q and E: rotate turtle

R: Reset window

Space: Turn pen on/off

Escape: Exit window


<b> Mouse Control:</b>

Left click: 

    place obstacle - define two corners
    
    move turtle - click turtle, then to target

Double click: remove obstacle
